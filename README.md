Using BLE connectivity from low cost low powered open hardware high resolution board. Many adjective for physiological goodness.  smartwatch (and other compatible devices) to stream physiological signals to computer.

Uses [bluepy](https://github.com/IanHarvey/bluepy). Clone this repo with the ` --recursive` flag then launch `make` inside `bluepy/bluepy`.

Requires [LSL](https://github.com/sccn/labstreaminglayer): install with `pip install pylsl`.
